# 新能源热利用与热发电期末刷题网站

这是一个本地静态刷题网页，直接读取 `data/questions/questions.js` 中已经整理好的题库数据。

## 打开方式

双击打开 `index.html`，或在浏览器中打开该文件。

## 当前功能

- 按题型筛选：全部、单选、多选、判断、填空、大题
- 搜索题目、答案、分类、选项
- 顺序刷题和乱序刷题
- 只看未完成
- 提交答案并判断正确/错误
- 查看答案和解析
- 错题自动记录、查看错题、从错题移除
- 收藏/取消收藏、查看收藏题
- 背答案模式，并可标记完成
- 已刷进度、错题、收藏按当前题库隔离保存，白天/夜间模式使用 localStorage 共用保存
- 支持简单上下标 HTML 标签，如 `<sub>`、`<sup>`

## 数据说明

题库文件位于：

`data/questions/questions.js`

本网页只读取该文件，不修改原始题库。
